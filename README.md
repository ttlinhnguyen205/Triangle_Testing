# Triangle Testing

## Mục tiêu

Bài làm thực hiện kiểm thử phương thức `classify(int a, int b, int c)` trong chương trình Java `Triangle`.

Nội dung bao gồm:

1. Vẽ Control Flow Graph (CFG).
2. Tính Cyclomatic Complexity (CC).
3. Liệt kê các đường đi độc lập.
4. Thiết kế test case đạt:
   - 100% Statement Coverage
   - 100% Branch Coverage

## Cấu trúc thư mục

Project được xây dựng theo cấu trúc Maven:

```text
TriangleTesting/
│
├── pom.xml
├── README.md
├── Report_Triangle_CFG_CC_Testing.md
│
└── src/
    ├── main/
    │   └── java/
    │       └── Triangle.java
    │
    └── test/
        └── java/
            └── TriangleTest.java
```
